## Getting Started

To explore the course content:

1. Browse the rendered HTML files in the repository (e.g., `Chapters.html`, `Mini_reports.html`, `Timetable.html`).  
2. Visit the published course website linked above.  
3. If you wish to build or modify documents locally, install R and RStudio, and open the `.Rproj` file in your local clone.

### Running R Markdown Documents

To render R Markdown files (e.g., to update a chapter draft):

1. Clone this repository locally.  
2. Open the `.Rproj` file in RStudio.  
3. Ensure required R packages are installed (as listed in the top of each Rmd file).  
4. Knit the document to HTML or PDF from within RStudio.
